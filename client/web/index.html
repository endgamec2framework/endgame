Requirements

        

          
ToolVersionPurpose

          
            
Go1.21+Compile server, client and Linux agent

            
gitanyClone repo and download BOFs

            
gcc-mingw-w64anyCross-compile Windows agent

            
mono-mcsanyExecute-assembly support (optional)

          
        

        
apt-get install -y golang git gcc-mingw-w64-x86-64 mono-mcs


        
Automatic installation

        
bash install.sh

        
The installer compiles everything, generates certificates and creates the operator profile stark.json.


        
First run

        
1Generate certificates and operator profile

        
./bin/c2-server &
./bin/c2-server new-operator -name stark -out stark.json
pkill c2-server

        
2Start everything at once

        
make start          # uses stark.json + port 8888 by default
make stop           # stop server and client

        
Logs at log/c2-server.log and log/c2-client.log


        
Remote access (SSH tunnel)

        
The operator port (31337) listens on loopback only. If the client runs on a different machine:

        
ssh -L 31337:127.0.0.1:31337 user@<server-ip>
./bin/c2-client -profile stark.json -gui-port 8888 -gui-only

      


      
      

        
Server and client

        

          
CommandDescription

          
            
make startStart server + GUI client in the background (requires compiled binaries)

            
make stopStop server and GUI client

            
make build-startCompile everything then start — equivalent to server + client + start

            
make allCompile server, client and Windows agent EXE

            
make serverCompile bin/c2-server only

            
make clientCompile bin/c2-client only

            
make runCompile server and run in foreground (logs to terminal)

          
        

        
make start options:

        
make start PROFILE=stark.json   # custom operator profile
make start GUI_PORT=9999        # alternate GUI port
make start GUI_HOST=0.0.0.0     # expose GUI on all interfaces


        
Agents

        

          
CommandDescription

          
            
make agent-exeWindows HTTP agent → bin/agent.exe

            
make agent-mtlsWindows mTLS agent → bin/agent-mtls.exe (requires make certs)

            
make agent-linuxLinux ELF agent → bin/agent-linux

            
make agent-darwinmacOS agent → bin/agent-darwin

            
make agent-rawConvert agent.exe to raw shellcode via donut

          
        

        
make agent-exe C2_HOST=10.2.20.200 SLEEP=30 JITTER=20
make agent-mtls C2_HOST=10.2.20.200


        
Utilities

        

          
            
make certsGenerate CA + server certificates (first time only)

            
make toolsCopy .NET tools into data/uploads/ (clones SharpCollection if not present)

            
make bofsClone / update BOF collections into bof/

            
make depsInstall system dependencies via apt-get

            
make initInitialise Go module and download dependencies

            
make cleanRemove binaries, database, certificates and uploads

          
        

      


      
      

        
Default ports

        

          
PortProtocolPurpose

          
            
8080HTTPAgent beacon (plain HTTP)

            
8443mTLSAgent beacon (mutual TLS)

            
443HTTPSAgent beacon (one-way TLS)

            
31337mTLS loopbackOperator API (127.0.0.1 only)

            
8888HTTPWeb GUI (default)

            
7777HTTPLocal file stager

          
        

        
All ports are configurable. The operator port (31337) is loopback-only — use an SSH tunnel for remote access.

      


      
      

        
Directory layout

        

          
            
bin/Compiled binaries (c2-server, c2-client, agents)

            
certs/CA and TLS certificates (auto-generated)

            
data/c2.dbSQLite database (agents, tasks, credentials…)

            
data/uploads/Files available to push to agents

            
data/downloads/Files pulled from agents

            
data/stager/Files served by the stager (port 7777)

            
bof/BOF collections installed with make bofs

            
agents/agent-go/Go agent source code

            
server/C2 server source code

            
client/Client source code (CLI + web GUI)

          
        

        
Logs

        

          
            
log/c2-server.logServer log (agent beacons, tasks, errors)

            
log/c2-client.logGUI client log (access token, errors)

# Ethical Use Policy — ENDGAME C2 FRAMEWORK

ENDGAME is a professional security tool developed for legitimate purposes. By downloading, cloning, or using this software you agree to the following terms.

---

## Permitted Use

- Authorized penetration testing engagements with explicit written permission from the system owner
- Internal red team operations within your own organization's infrastructure
- Security research in isolated lab environments and sandboxes
- Capture The Flag (CTF) competitions and training exercises
- Academic research and educational demonstrations
- Development and testing of defensive security controls (EDR, SIEM, SOAR)

---

## Prohibited Use

- Accessing, testing, or attacking systems without explicit written authorization
- Any activity that violates local, national, or international law
- Targeting critical infrastructure, healthcare systems, or government networks without proper authorization
- Distribution of compiled agents to unauthorized parties
- Use in any activity intended to cause harm, financial loss, or disruption

---

## Responsible Disclosure

If you discover a vulnerability in ENDGAME itself, please report it responsibly:

1. **Do not** open a public GitHub issue with exploit details
2. Contact the maintainers privately before public disclosure
3. Allow reasonable time for a fix before publishing findings

---

## Professional Standards

Users of this tool are expected to operate under recognized professional standards:

- [PTES — Penetration Testing Execution Standard](http://www.pentest-standard.org)
- [OWASP Testing Guide](https://owasp.org/www-project-web-security-testing-guide/)
- [EC-Council Code of Ethics](https://www.eccouncil.org/code-of-ethics/)
